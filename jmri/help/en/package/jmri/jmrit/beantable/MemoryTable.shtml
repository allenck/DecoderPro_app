The Memory Table

You can use the Memory Table to look at the contents of JMRI memory variables.

Note that you can sort the table by each of the columns, just click on the column heading.

To change the value stored in a particular memory, just click on the value column in the desired row. Enter a new value, followed by hitting the return or enter key to store the value.

To create a new memory variable, click on the "Add..." button at the bottom of the window.

For more information on memories, see the memory help page.
