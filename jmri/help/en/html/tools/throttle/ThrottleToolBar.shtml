Throttle Tool Bar



The Throttle toolbar is enablable from the Throttles Preferences Pane.

When its enabled, a Throttle Window can host a set of Throttles, these Throttles are browsable using the:

Previous Button

Move to previous Throttle for that Throttle Window.

Next Button

Move to next Throttle for that Throttle Window.

New Button

Add a new Throttle in that Throttle Window.

Once the number of Throttles in a Throttle Window is greater than one, Next, Previous and Trash buttons are enabled.

Trash Button

Remove the curent Throttle from that Throttle Window.

When the number of Throttles in a Throttle Window is equal to one, Next, Previous and Trash buttons are disabled.

Stop Button

Send a stop message to all throttles managed by JMRI.

Power Button

Allows for layout power control if possible (red is off, green is on, yellow is status unknown).

List Button

Show/Hide the throttles list Window.
