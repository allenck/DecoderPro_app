JMRI: Marklin Command Station 2

JMRI can connect to a Marklin Command Station 2 using the built in ethernet interface.
It is possible for multiple PCs to connect to the CS2 using IP/UDP.

The CS2 uses its own CAN BUS protocol running over UDP, responses from the CS2 are sent out as a broadcast message and therefore the PCs connecting to the CS2 must be in the same IP subnet otherwise the reply messages will not get seen.

This implementation of the CS2 CAN protocol was developed and tested on firmware version 2. This interface is still under development if you find any issues then please raise a issue on the yahoo group

To configure the connection:

  • Open the JMRI "Preferences" under the "Edit" menu,
  • Select "MARKLIN" as the manufacuturer, then the connection type is Command Station 2,
  • Enter in the address assigned to the CS2.
  • Click "Save".

Turnouts, Sensors, Throttle and power support is available for a JMRI-connected CS2 unit.

Throttle support includes the ability to select a specific protocol (Marklin, DCC & mfx)

Consisting, and Programming are not yet support.

Support for the Marklin CS2 was first available in JMRI 3.1.1.
