DecoderPro User Guide - Roster File Details

The roster information presented in the selection box, and used for automatic selection of a locomotive, is stored in the roster.xml file in the "prefs" directory. You can open this file and look at it to see the format. You should use the Roster menu to manipulate it to ensure that it stays self-consistent.

To move the entire roster to another computer, first find the preferences directory using the information on the "Configuration Files" page.

In the preferences directory there will be a file called "roster.xml", and a directory called "roster". You need to move the "roster.xml" file, the "roster" directory, and all it's contents.
