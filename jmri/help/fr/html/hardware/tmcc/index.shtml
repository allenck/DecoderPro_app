Guide Matériel JMRI: Lionel TMCC

Système supporté

(Lionel TMCC ou Lionel universel à travers le port série sur la centrale de base.)

Installation

D'abord, connecter le port série de votre ordinateur à la centrale de commande TMCC en utilisant un câble série. Cela devrait être un câble "droit" ou "modem" , pas un câble "croisé" ou "null-modem". Si votre ordinateur ne dispose pas d'un port série intégré, vous pouvez utiliser un adaptateur USB-série, mais assurez-vous d'installer les bon conducteurs.

Ensuite, configurez le programme.

  • Commencez avec votre programme basé sur JMRI (DecoderPro, PanelPro, etc.)
  • Allez dans le panneau des préférences du programme.
  • Celui-ci s'ouvre automatiquement la première fois qu'un programme est exécuté, ou vous pouvez le sélectionner dans le menu "édition".
  • Sélectionnez "TMCC (Lionel)" dans la boite de sélection du haut.
  • Dans la deuxième boîte de sélection, choisissez le nom du port série auquel vous avez connecté au système TMCC.
  • Si la vitesse de transmission ne montre pas une valeur fixe, réglez la à 9600 bauds.
  • La série de boutons radio étiquetés "style graphique" vous permet de sélectionner la façon dont le programme va apparaitre. "Metal" est le thème natif de Java et chercher ceux disponible sur tous les types. D'autres choix peuvent varier avec votre type d'ordinateur. "Motif", "Windows", "MacOS" et "MacOS adaptative" apparence native sous Linux, Windows, MacOS "Classic" et Mac OS X respectivement.
  • Dans la boîte de sélection en bas, vous pouvez choisir l'option par défaut du format du programmateur. Vous pouvez remplacer celui ci à chaque fois que vous ouvrez le programmateur, ce n'est donc ici que par commodité.
  • Cliquez sur "Enregistrer". Il vous sera demandé si vous voulez quitter le programme, cliquez sur "Oui".
  • Redémarrez le programme. Vous devriez être en place et fonctionnel.
