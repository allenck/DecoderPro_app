Installations

  1. Pour connecter votre ordinateur à un système Zimo MX-1 , vous pouvez normalement connecter la centrale au port série de l'ordinateur à l'aide d'un câble série.
  2. Puis allez dans le panneau des préférences d'un programme basé sur JMRI. Celui ci s'ouvre automatiquement la première fois qu'un programme est exécuté, ou vous pouvez le sélectionner dans le menu "édition".
  3. sélectionnez "Zimo MX-1" de la boîte de sélection en haut. Vous pouvez alors sélectionnez le port série approprié dans la seconde boîte de sélection .
  4. La boîte de sélection de vitesse de transmission( Baud/Rate)vous donnera des choix de 1200 à 38.400 bauds. Sélectionner la vitesse qui est configurée dans votre MX-1.
  5. La série de boutons radio étiquetés "style GUI" vous permet de sélectionner la façon dont le programme va s'afficher. "métal" est le mode natif Java. Voir pour d'autres choix possibles en fonction de votre type d'ordinateur. "motif", "fenêtres", "MacOS" et "MacOS adaptative" apparence native sous Linux, Windows, MacOS "classique" et Mac OS X respectivement.
  6. dans la case de sélection en bas, vous pouvez choisir l'option par défaut du format de programmeur. Vous pouvez le remplacer à chaque fois que vous ouvrez le programmateur, ce n'est donc ici juste que par convenance.
  7. Cliquez sur "Enregistrer". Il vous sera demandé si c'est Ok pour quitter le programme , cliquez sur "oui".
  8. Redémarrer le programme. Vous devriez être prêt à fonctionner.
