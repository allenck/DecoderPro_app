JMRI Guide d'installation: Connexions port série MacOS

Le ports série Macintosh utiliser un connecteur "mini-DIN-8", qui ne contient pas tous les signaux d'un connecteur RS232 typiques Cette page explique comment câbler un adaptateur de sorte qu'un Macintosh puissent travailler avec JMRI et/ou un LocoBuffer ou LocoBuffer II. Le même adaptateur peut également travailler avec d'autres protocoles, en particulier ceux qui utilisent le "contrôle de flux matériel".

Une bonne étude de connexions Macintosh série, y compris les bonnes techniques de débogage, est disponible à http://francis.courtois.free.fr/jc1/serial/main.html. lien cassé le 2013-09-09.

Notez que le Macintosh ne possède qu'une seule sortie d'état et d'entrée.

Si le périphérique série utilise le "contrôle de flux matériel", il entend faire valoir que le fil CTS indique quand il est OK pour envoyer des données. Cela doit être connecté au fil d'entrée du Macintosh, ou le programme ne sera pas en mesure d'envoyer.

Le statut sortie sera activé par JMRI au démarrage. Il devrait être connecté à n'importe quel fil de statut (par exemple RTS) que le périphérique série surveille.
