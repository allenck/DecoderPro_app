JMRI: Panneau Servlet

Les Servlets Panel JMRI offrent un accès Web à des informations sur les panneaux JMRI "ouvert". Ce servlet répond à des modèles d'URL suivants:
    :
    • /panel/Panel/<panel name>
    • /panel/ControlPanel/<panel name>
    • /panel/Layout/<panel name>
    où Panneau, ControlPanel et Réseau se réfèrent à l'éditeur JMRI utilisé pour créer chaque type de panneau.

    Le servlet renvoie un flux xml avec divers éléments nécessaires pour recréer ce panneau particulier externe.

    Actuellement, il s'agit d'une application web côté client appelé "/panel" qui utilise ces définitions "dessiner" des panneaux dans un navigateur, ce qui permet des clics sur les éléments actifs et mettre à jour des modifications apportées par cette application et ailleurs dans JMRI.
