JMRI: Liste Servlet

La Liste Servlet JMRI fournit un accès Web à des informations sur les entrées liste JMRI. Ce servlet répond aux modèles d'URL suivants:
    :
    • /roster/
    • /roster/list
    • /roster/list?filter=filter[&filter=filter]
    • /roster/<ID>
    • /roster/entry/<ID>
    • /roster/<ID>/image
    • /roster/<ID>/icon
