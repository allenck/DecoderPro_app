Tachymètre

Le compteur de vitesse mesure le temps d'un capteur à un autre, puis calcule la vitesse en fonction de la distance parcourue et du temps mis pour le faire.

Entrez un capteur Sysname dans le champs "démarre minuteries sur". Puis entrer dans un autre capteur Sysname dans le champs "arràte le Chrono 1 sur". Mesurer la distance entre les deux, convertir en pieds pour votre échelle et entrez cela dans le champs "Distance 1".

La vitesse de votre train entre les deux capteurs sera affichée dans le champ "Vitesse Chrono 1".

Vous pouvez ajouter un troisième capteur dans le champs "chronomètre deux s'arràtera deux sur" et qui donnera la vitesse entre le capteur de départ et que le capteur dans le champ "Timer 2 Vitesse . Vous avez également besoin d'entrer la valeur dans le champ "Distance 2"'en pieds à l'échelle entre le capteur de départ et ce capteur.

Vous pouvez choisir de déclencher la minuterie en entrée/sortie du capteur avec les boutons "Entrée" ou de "Sortie". Les indicateurs sur le côté droit du volet sont l'état des capteurs que vous avez sélectionnés.
