Le Tableau Rapport

Les "Rapports" JMRI montrent les informations d'état reçues de votre réseau. Par exemple, transpondeur Digitrax fournit des informations via des objets Rapport JMRI. Le tableau Rapport montre tous les Rapports existants et de leurs plus récentes informations.

Pour plus d'informations sur les Rapport et leur utilisation, consultez la page principale Rapport.
