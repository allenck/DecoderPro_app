Le Tableau Logique MâtSignal

Une logique MâtSignal, se compose d'un Mât de destination, un ou plusieurs aiguillages, canton, de capteurs ou d'autres MâtsSignal qui doivent être dans un état précis avant le MâtSignal source, peut être réglé à partir d'un aspect "Danger".

Si vous avez dessiné correctement votre réseau sur l'éditeur de réseau avec toutes les informations de cantons, il est possible pour le système de construire automatiquement toutes les paires de MâtSignal source et destination, avec les cantons, les aiguillages et leurs paramètres.

Colonnes du Tableau Logique MâtSignal

Source
C'est le mât de signal que vous souhaitez mettre en place au début de la logique. Il peut y avoir plusieurs instances du mât source dans la table, selon le nombre de mâts de destination différents qui sont accessibles depuis ce mât.
Considérez cela comme le mât de signal auquel le conducteur est actuellement confronté.
Apparence
L'aspect actuel du mât source
Destination
C'est le mât de signalisation de destination qui est le prochain sur la ligne du mât source. Il peut y avoir plusieurs instances du mât de destination, tous avec des mâts source différente.
Apparence
L'aspect actuel du mât de destination
Commentaire
Un champ libre pour l'utilisateur sous forme modifiable par l'utilisateur pour attribuer des commentaires sur la la logique
Supprimer
Appuyer le bouton "Supprimer" supprimera la logique du mât de signalisation du tableau.

Notez que cela ne supprime pas la logique mât de signalisation de tous les fichiers que vous avez sauvegardés qui la contiennent. Si vous chargez un de ces fichiers, peut-être lorsque vous redémarrez le programme, l'itinéraire va réapparaître. Vous devez utiliser l'une des options "Stocker dans Fichier" dans les menu "Fichier" pour enregistrer de nouvelles versions de tous les fichiers qui contiennent cette voie.

Activé
Pour désactiver une Logique de Mât de Signalisation depuis un déclenchement, décochez la case "Activé" pour l'itinéraire.
Modifier
Appuyez sur ce bouton pour modifier la Logique du Mât de Signalisation. Pour plus d'informations, voir: Ajout/édition Logique Mât de Signalisation.

Ajouter Logique Mât de Signalisation

Cela se fait via le bouton Ajouter au bas de l'écran, pour plus d'informations sur le montage et l'ajout de Logique Mât de Signalisation voir Ajout/édition Logique Mât de Signalisation.
