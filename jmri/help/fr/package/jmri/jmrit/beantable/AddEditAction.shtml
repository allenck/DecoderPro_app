Ajouter/éditer Action

La fenêtre Ajouter/Modifier Action

Les Actions sont des tâches qui peuvent être lancées automatiquement comme un train qui traverse en mode automatique le réseau. La fenêtre Ajouter/Modifier action (voir ci-dessous pour la version"Ajouter") prévoit la création d'actions nouvelles et pour l'édition des actions en cours. La version «Modifier» de cette fenêtre est la même que celle montrée ci-dessous, à l'exception du bouton Créer Nouvelle action qui est est remplacé par un bouton Mise à Jour Actionet les sélections et données sont remplies par les valeurs actuelles de l'action en cours d'édition.

Une action est définie en spécifiant Quelle action est demandée et Quandl'action doit être engagée.

Pour spécifier la partie Quand, sélectionnez une option dans le menu contextuel Quand, et remplir toutes les données qui pourraient être nécessaires. Les Champs d'élément de données ou de menus apparaîtront lorsque l'élément qui en a besoin est sélectionné. Chaque élément de données est fourni avec une description textuelle (bulle d'aide) qui peut être appelée en pointant le curseur sur l'élément. Chaque option Quand est décrite plus en détail dans Actions.

Un retard optionnel, spécifié en millisecondes, est disponible pour chaque option Quand . Aucun retard est un défaut. Si un délai optionnel n'est pas entré (ou un temporisation zéro est entrée), l'action sera lancée immédiatement après le quand l'événement se soit produit. Si un délai est spécifié, le programme attendra pendant le temps spécifié après le quand l'événement se soit produit avant d'initier l'action.

Pour spécifier Quelle action est demandée, sélectionner une option dans le menu contextuel Quelle , et remplir toutes les données qui pourraient être nécessaires. Les Champs d'élément de données apparaîtront lorsque l'élément qui en a besoin est sélectionné. Chaque élément de données est fourni avec une description textuelle (bulle d'aide) qui peut être appelée en pointant le curseur sur l'élément. Chaque Quelle action est décrite plus en détail dans Actions.

Quand l'action est complètement définie cliquez sur le bouton Créer une Nouvelle Action (ou le bouton Mise à Jour Action si édition en cours). Toutes les données nécessaires seront validées. Si un problème est détecté, un message apparaîtra, et la fenêtre Ajouter/éditer Actions restera pour permettre la correction du problème. Si tout est vérifié, la fenêtre Ajouter/éditer action va disparaître et les actions(nouvelles ou modifiées) apparaîtront dans la fenêtre Voir Actions.

Cliquez sur le bouton Annuler pour refermer la fenêtre Ajouter/Modifier l'Action, sans créer une nouvelle action ou sans achever le montage d'une action existante . La fermeture de la fenêtre Ajouter/Modifier l'Action équivaut à cliquer sur le bouton Annuler.
