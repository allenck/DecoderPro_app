Configurer Digitrax BDL168

L'outil de programmation JMRI BDL168 vous permet de configurer les options internes d'un BDL168 directement à partir de votre ordinateur.

Pour ouvrir cet outil à partir de la page de démarrage JMRI sélectionnez l'onglet LocoNet puis trouver l'outil sur la liste déroulante.

Pour l'utiliser, vous mettez l'adresse d'une carte BDL168 dans le champ texte et cliquez sur "Lire BDL168". L'outil relira l'état actuel des différentes options et définira les cases à cocher pour correspondre à leurs valeurs actuelles.

Vous pouvez alors, si vous le souhaitez, modifier les cases et cliquez sur "Ecrire dans DS64" pour rendre vos modifications permanentes.

Limitations

En raison de la façon dont fonctionne la carte BDL168, cet outil ne peut pas changer l'adresse de base de l'unité.
