Configurer Digitrax DS64

L'outil de programmation JMRI DS64 vous permet de configurer les options internes d'une DS64 directement à partir de votre ordinateur.

Pour ouvrir cet outil à partir de la page de démarrage JMRI sélectionnez l'onglet LocoNet puis de trouver l'outil sur la liste déroulante.

Pour l'utiliser, vous mettez l'adresse d'une carte DS64 dans le champ texte et cliquez sur "Lire depuis DS64". L'outil relira les état actuels des différentes options et définira les cases à cocher pour correspondre à  leurs valeurs actuelles.

Vous pouvez alors, si vous le souhaitez, modifier les cases et cliquez sur "Écrire dans DS64" pour rendre vos modifications permanentes.

Limitations

En raison de la façon dont fonctionne la carte DS64, cet outil ne peut pas changer l'adresse de base de l'unité.

A l'heure actuelle, les itinéraires DS64 ne peuvent pas être programmé par cet outil. C'est possible, mais personne n'a écrit le code pour le moment car les itinéraires JMRI sont plus puissant et facile à utiliser
