Statistiques LocoNet

Les Interfaces avancées LocoNet comme la série des LocoBuffer RR-CirKits ou la série de PRn Digitrax collectent des statistiques sur le fonctionnement de LocoNet qui lui est reliée. Cette fenêtre les affiche.

Le bouton "Update" demande des informations à l'interface pour l'affichage.

Valeurs LocoBuffer

jmri/jmrix/loconet/locostats/LocoStatsFrame jmri/jmrix/loconet/locostats/LocoStatsFrame
Version
Le numéro de version de l'appareil.
Breaks
Le nombre de ruptures de LocoNet vu depuis la dernière fois que l'interface a été allumée.
Erreurs
Le nombre d'erreurs LocoNet vu depuis la dernière fois l'interface a été allumée.

Digitrax PRn

Numéro de Série
Le numéro de série de l'appareil.
Statut PR2
Les bits d'état à partir d'une interface dans PR2 (programmateur) mode.
Courant
Le courant actuellement consommé par la voie, dans certaines unités arbitraires.
Version Matériel
Le numéro de version du matériel de l'interface.
Version du logiciel
Le numéro de version du logiciel de l'interface.
Compteur Message Bon
Le nombre de messages LocoNet bon vu depuis la dernière fois que l'interface a été allumée.
Compteur Message Mauvais
Le nombre de messages LocoNet mauvais vu depuis la dernière fois que l'interface a été allumée.
Statut MS100
Les bits d'état à partir d'une interface dans le mode MS100 (adaptateur LocoNet ) .
Certains d'entre eux sont affichées lorsque l'interface agit comme un programmateur (PR2 mode), et d'autres où il agit comme une interface (MS100 mode).
