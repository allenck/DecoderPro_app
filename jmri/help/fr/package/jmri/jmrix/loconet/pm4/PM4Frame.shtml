Configurer PM4 Digitrax

->

L'outil de programmation JMRI PM4 vous permet de configurer les options internes d'un PM4 directement à partir de votre ordinateur.

Pour ouvrir cet outil à partir de la page de démarrage JMRI sélectionnez l'onglet LocoNet puis trouver l'outil sur la liste déroulante.

Pour l'utiliser, vous mettez l'adresse d'une carte PM4 dans le champ texte et cliquez sur "Lire PM4». L'outil relira les états actuels des différentes options et définira les cases à cocher pour correspondre à leurs valeurs actuelles.

Vous pouvez alors, si vous le souhaitez, modifier les cases et cliquez sur "Ecrire dans DS64" pour rendre vos modifications permanentes.

Limitations

En raison de la façon dont fonctionne la carte PM4, cet outil ne peut pas changer l'adresse de base de l'unité.
