Envoyer Trame CAN


L'outi"Envoyer Trame CAN " vous permet d'envoyer directement des Trames CAN.

Il fonctionne mieux en combinaison avec "Afficher des données brutes" dans la Console CBUS, parce que les paquets sont entrés ici en rangées au format hexadécimal .

La section supérieure de la fenêtre vous permet d'envoyer une trame unique. Il suffit de taper le contenu du paquet de données en tant que nombres hexadécimaux, par exemple "83 7C". Si vous voulez spécifier le contenu en-tête, faites précéder le paquet de données avec l'en- tête dans "()" Caractères ou «[]» caractères pour un en-tête standard ou étendu, respectivement. Vous pouvez également entrer les paquets de format d'adresse CBUS, par exemple, "+ n4e1". Une fois que vous avez entré le paquet, cliquez sur "envoyer".

La section du bas vous permet de saisir une à quatre trames qui seront envoyé à plusieurs reprises dans l'ordre, avec des retards définis par l'utilisateur entre deux envois. Appuyez sur le bouton "Go" pour démarrer la séquence de transmission, et appuyez de nouveau pour arrêter la séquence.Send CAN Frame
