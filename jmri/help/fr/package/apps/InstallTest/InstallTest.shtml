JMRI: Fenêtre Principale InstallTest

La fenêtre principale InstallTest est destinée à montrer si vous avez Java et JMRI correctement installé. Vous ne devriez pas l'utiliser pour une activité normale.

Si vous avez obtenu cette page en exécutant et en sélectionnant Aide InstallTest, votre installation JMRI est fondamentalement OK. Vous pouvez passer à la configuration et l'éxecution de DecoderPro, PanelPro, etc
